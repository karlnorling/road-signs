## Why Road Signs?

Road signs are legally mandated, precisely specified, and used in safety-critical contexts. This
library packages official signs as accessible, fully typed components ready to drop into any
project — one npm package per country, so you only ship what you need.

<CardGrid>
  <Card title="Per-country packages" icon="puzzle">
    Install only the countries you need. `@road-signs/us` for MUTCD, `@road-signs/uk` for TSRGD,
    and so on. Each package is self-contained with SVGs, raster assets, and a TypeScript registry.
  </Card>
  <Card title="React, Vue 3 & Web Components" icon="laptop">
    Generic framework adapters work with signs from any country package. One `<RoadSign>` component,
    any country's data.
  </Card>
  <Card title="CSS & SVG sprite" icon="seti:css">
    Each country package ships `sprite.svg` and `sprite.css` in its dist — use any sign with a
    single class, zero JavaScript.
  </Card>
  <Card title="Accessible by default" icon="approve-check-circle">
    Every component renders with `role="img"`, `aria-labelledby`, `<title>`, and `<desc>` inside
    the SVG.
  </Card>
</CardGrid>
